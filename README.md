DeepWon] A custom ML packages based on Tensorflow 2.0
-----------------------------------------------------

This repo contains various Deep Learning architectures, optimizers, schedulers and etc.   
Its directory structure is as follows.
* DeepWon
    * callbacks
        * callbacks.py
    * layers
        * layers.py
    * optimizers
        * optimizers.py
    * preprocess
        * image.py
        * tabular_data.py
    * schedules
        * schedulers.py
    * utils
        * utils.py
